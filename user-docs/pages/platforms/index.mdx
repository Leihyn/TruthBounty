## Import Cooldowns

<Callout type="warning">
  Each platform has a **24-hour cooldown** between imports. This prevents spam and ensures data quality.
</Callout>

Plan your imports accordingly - you can import from different platforms at the same time, but need to wait 24 hours to re-import from the same platform.
