| **Bets** | Total number of predictions |

## Navigation Menu

The main menu gives you access to:

### Dashboard
Your personal stats and import tools (you are here)

### Leaderboard
Global rankings of all TruthBounty users. Filter by:
- Platform
- Time period
- Tier

### Markets
View active prediction markets across platforms

### Copy Trading
Follow top traders or manage your followers

### Profile
Public view of your profile (what others see)

## Page Actions

### Refresh Stats
Click the refresh icon to update your stats with the latest data.

### Share Profile
Get a link to share your TruthBounty profile on social media.

### Export Data
Download your prediction history as a CSV file.

## Mobile Experience

On mobile, the dashboard adapts:
- Stats stack vertically
- NFT appears at the top
- Swipe between platform tabs
- Bottom navigation for quick access

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `D` | Go to Dashboard |
| `L` | Go to Leaderboard |
| `C` | Go to Copy Trading |
| `I` | Open Import Modal |
| `?` | Show all shortcuts |
