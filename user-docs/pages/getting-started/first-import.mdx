# Your First Import

Importing predictions is how you build your TruthScore. This guide walks you through importing from your first platform.

## How Importing Works

1. You select a platform (like PancakeSwap)
2. TruthBounty fetches your prediction history
3. You review and confirm the data
4. Your TruthScore is calculated and updated
5. Your NFT metadata updates automatically

## What Gets Imported

When you import, TruthBounty records:

| Data | Description |
|------|-------------|
| **Win Count** | Number of successful predictions |
| **Total Bets** | Total number of predictions made |
| **Volume** | Total amount wagered (in platform currency) |
| **Batch Hash** | Unique identifier to prevent duplicate imports |

<Callout type="info" emoji="üîí">
  **Privacy:** We only import aggregate statistics, not individual trade details.
</Callout>

## Import Rules

### Cooldown Period

You can only import from each platform **once per hour**. This prevents spam and ensures fair scoring.

### No Duplicate Imports

Each set of predictions can only be imported once. The system uses a "batch hash" to track this.

### Accuracy Verification

TruthBounty verifies data directly from the blockchain, so:
- ‚úÖ You can't fake your history
- ‚úÖ Everyone's reputation is trustworthy
- ‚úÖ The leaderboard is fair

## Troubleshooting

### "No predictions found"

**Possible causes:**
- You're using a different wallet than you trade with
- You haven't made any predictions on this platform yet
- The platform data is still syncing

**Solution:** Double-check you're connected with the correct wallet address.

### "Cooldown active"

You've imported from this platform recently. Wait until the cooldown ends (shown on the button) before importing again.

### "Import failed"

This usually means a network issue. Try:
1. Wait a few seconds and try again
2. Check your internet connection
3. Refresh the page
